# HydraTutorial



## Installation
```
git clone https://github.com/AutoML.org/HydraTutorial.git
cd HydraTutorial
conda create -n hydra_tutorial python=3.8
conda activate hydra_tutorial

# Install for usage
pip install .

# Install for development
make install-dev
```
